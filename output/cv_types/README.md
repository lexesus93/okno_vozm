# Типовые CV-пакеты

Эта папка содержит 3 переиспользуемых типа CV, собранных из уже проработанных кейсов откликов.

## Типы

- `01-data-ai-platform-leader` — Data & AI Platform Leader / CDO-CTO.
- `02-ai-data-business-adoption-partner` — AI/Data Business Adoption Partner.
- `03-professional-it-consulting` — Professional IT Consulting / Data & AI Transformation.

## Состав каждой папки

- `analysis.md` — логика типа и соответствие кейсам.
- `skills_requirements.md` — навыки, требования, маркеры, риски.
- `cover_letter_template.md` — шаблон сопроводительного письма.
- `tailored_cv.md` — самостоятельная типовая версия CV.
- `interview_prep.md` — подготовка к интервью.
- `source_cases.md` — карта исходных кейсов и тезисов.
